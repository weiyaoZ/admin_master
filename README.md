# 后台管理项目

## 介绍
该项目为后台管理项目，基于elementUI 搭建而成


## 安装教程
```
### 安装依赖
1.  npm install
```
```
### 运行项目
2.  npm run serve
```
```
### 打包项目
3.  
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```
